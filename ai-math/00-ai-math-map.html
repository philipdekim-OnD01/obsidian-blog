00_AI_수학_전체지도

한 문장 요약

AI 수학은 데이터를 숫자로 표현하고, 함수로 예측하고, 손실을 줄이도록 파라미터를 바꾸는 과정을 이해하기 위한 도구다.

왜 AI에 필요한가

AI 모델은 겉으로는 복잡해 보이지만 기본 구조는 단순하다.

입력 데이터 x -> 모델 f(x; θ) -> 예측값 y_hat -> 손실 L(y_hat, y) -> 파라미터 θ 업데이트

여기서 수학은 각각의 부분을 설명한다.

AI 구성 요소 수학 분야 핵심 역할
데이터 표현 선형대수 벡터, 행렬, 텐서로 데이터를 표현
모델 계산 함수와 선형대수 입력을 출력으로 바꾸는 계산
오차 측정 확률, 통계, 정보이론 예측이 얼마나 틀렸는지 측정
학습 방향 미적분 파라미터를 어느 방향으로 바꿀지 계산
학습 과정 최적화 손실을 줄이는 업데이트 규칙

AI 모델을 함수로 보기

수학에서 함수는 입력을 받아 출력을 만드는 규칙이다.

y = f(x)

AI 모델도 같은 방식으로 볼 수 있다.

예측값 = 모델(입력 데이터)

예를 들어 집값 예측 모델은 다음처럼 볼 수 있다.

집의 크기, 위치, 방 개수 -> 모델 -> 예상 집값

이미지 분류 모델은 다음처럼 볼 수 있다.

이미지 픽셀값 -> 모델 -> 고양이일 확률, 개일 확률, 자동차일 확률

언어모델은 다음처럼 볼 수 있다.

지금까지의 문장 -> 모델 -> 다음 단어 또는 다음 토큰의 확률

파라미터란 무엇인가

파라미터는 모델 안에서 학습을 통해 바뀌는 숫자다.

가장 단순한 모델은 일차함수다.

y_hat = wx + b

여기서 각 기호의 의미는 다음과 같다.

기호 의미 AI 관점
x 입력 데이터
y_hat 예측값 모델의 출력
w 기울기 weight, 가중치
b 절편 bias, 편향

AI가 학습한다는 말은 대부분 w, b 같은 파라미터 값을 조금씩 바꿔서 더 좋은 예측을 하도록 만든다는 뜻이다.

손실함수란 무엇인가

손실함수는 예측이 실제 정답과 얼마나 다른지 숫자로 나타내는 함수다.

예를 들어 실제값이 10이고 예측값이 7이면 오차는 3이다.

오차 = 예측값 - 실제값

오차가 음수일 수도 있으므로 보통 제곱해서 쓴다.

손실 = (예측값 - 실제값)^2

이것이 평균제곱오차, 즉 MSE의 기본 아이디어다.

L = (y_hat - y)^2

AI 학습의 목표는 손실함수 L을 작게 만드는 것이다.

학습은 무엇을 하는 과정인가

학습은 다음 과정을 반복한다.

  1. 현재 파라미터로 예측한다.
  2. 예측값과 실제값을 비교해 손실을 계산한다.
  3. 손실을 줄이는 방향을 미분으로 찾는다.
  4. 파라미터를 조금 바꾼다.
  5. 손실이 충분히 작아질 때까지 반복한다.

이를 간단히 쓰면 다음과 같다.

예측 -> 손실 계산 -> 기울기 계산 -> 파라미터 업데이트

수학 분야별 위치

1. 선형대수

데이터와 모델 계산을 벡터와 행렬로 표현한다.

Y = XW + b

AI에서 선형대수가 필요한 이유:

  • 이미지, 음성, 문장을 숫자 배열로 표현한다.
  • 여러 데이터를 한 번에 계산한다.
  • 신경망의 각 층을 행렬곱으로 표현한다.
  • 임베딩 공간에서 의미의 거리와 방향을 다룬다.

2. 미적분

손실이 줄어드는 방향을 찾는다.

w_new = w_old - learning_rate * gradient

AI에서 미적분이 필요한 이유:

  • 손실함수의 기울기를 계산한다.
  • 각 파라미터가 손실에 미치는 영향을 계산한다.
  • backpropagation을 이해한다.
  • optimizer의 업데이트 원리를 이해한다.

3. 확률과 통계

불확실성과 데이터의 패턴을 다룬다.

AI에서 확률과 통계가 필요한 이유:

  • 분류 결과를 확률로 해석한다.
  • 데이터 분포를 이해한다.
  • 평균, 분산, 정규화의 의미를 이해한다.
  • train, validation, test 성능 차이를 해석한다.

4. 정보이론

예측 확률분포가 정답 분포와 얼마나 다른지 측정한다.

AI에서 정보이론이 필요한 이유:

  • entropy와 cross entropy를 이해한다.
  • 분류 모델의 loss를 이해한다.
  • 언어모델의 다음 토큰 예측을 확률분포로 이해한다.

더 나은 예측

예제: 가장 작은 AI 모델

데이터가 다음과 같다고 하자.

x y
1 2
2 4
3 6

이 데이터는 대략 다음 규칙을 가진다.

y = 2x

하지만 모델은 처음에 이 규칙을 모른다.

y_hat = wx + b

처음 파라미터를 이렇게 잡았다고 하자.

w = 1
b = 0

그러면 x = 3일 때 예측값은 다음과 같다.

y_hat = 1 * 3 + 0 = 3

실제값은 6이므로 손실은 다음과 같다.

L = (3 - 6)^2 = 9

학습은 w를 1에서 2에 가깝게 바꾸는 과정이다. 즉, 모델이 데이터 속 규칙을 숫자로 찾아가는 과정이다.
