OnDevice AI 경량화 강의: RPS QAT 초보자용 코드

이 강의자료는 RPS_QAT_DenseNet121_초보자용.ipynb를 기준으로, 라즈베리파이 같은 온디바이스 환경에서 왜 모델 경량화가 필요한지와 QAT가 어떤 역할을 하는지를 설명한다.

핵심 목표는 “정확도 높은 CNN 모델을 작은 장치에서 실행 가능한 TFLite 모델로 바꾸는 과정”을 이해하는 것이다.

1. 경량화가 필요한 이유

PC나 GPU 서버에서는 큰 CNN 모델도 실행할 수 있다. 하지만 라즈베리파이, 마이크로컨트롤러, 카메라 모듈 같은 온디바이스 환경에서는 조건이 다르다.

  • 메모리가 작다.
  • CPU 성능이 제한적이다.
  • 배터리 또는 발열 제약이 있다.
  • 네트워크 없이 로컬에서 추론해야 할 수 있다.
  • 실시간 반응 속도가 중요하다.

그래서 온디바이스 AI에서는 “정확한 모델”만으로는 부족하다. 모델 크기, 추론 속도, 메모리 사용량까지 같이 봐야 한다.

2. 오늘 코드의 전체 흐름

1. RPS_Dataset 자동 로딩
2. 이미지를 64 x 64 x 3 배열로 변환

4. ImageNet 사전학습 DenseNet121 불러오기
5. 가위/바위/보 3개 클래스로 분류기 변경
6. QAT 적용
7. 가장 좋은 모델 저장
8. TFLite 모델로 변환
9. 라즈베리파이 배포용 .tflite 파일 생성

3. 데이터 로딩

학생용 노트북은 Google Drive나 zip 파일을 사용하지 않는다. 저장소 안의 RPS_Dataset 폴더를 자동으로 찾는다.

RP2-RPS-QAT-Lab/
  RPS_Dataset/
    0/ scissors
    1/ rock
    2/ paper
  notebooks/
    RPS_QAT_DenseNet121_초보자용.ipynb

GCP GPU 서버와 VSCode에서는 git rev-parse --show-toplevel로 저장소 루트를 찾고, 그 아래의 RPS_Dataset을 사용한다. 따라서 학생이 저장소를 fork하고 clone하면 별도 데이터 다운로드 없이 실행할 수 있다.

4. 데이터 보강

손 모양 이미지는 촬영 위치, 회전, 밝기, 크기에 따라 달라진다. 데이터 보강은 학습 중 이미지를 조금씩 변형해서 모델이 더 일반적인 특징을 배우게 하는 방법이다.

보강 방식 의미 효과
Rotation 이미지를 약간 회전 손 각도 변화에 강해짐
Shift 이미지를 좌우/상하 이동 손 위치 변화에 강해짐
Zoom 이미지를 확대/축소 카메라 거리 변화에 강해짐
Flip 좌우 반전 손 방향 변화에 강해짐

5. DenseNet121을 사용하는 이유

DenseNet121은 ImageNet으로 미리 학습된 CNN 모델이다. 처음부터 가위바위보 이미지만으로 CNN을 학습하는 것보다, 이미 일반 이미지 특징을 배운 모델을 가져와 마지막 분류기만 바꾸는 방식이 효율적이다.

base_model = keras.applications.DenseNet121(
    weights="imagenet",
    include_top=False,
    input_tensor=x
)

include_top=False는 ImageNet의 1000개 클래스 분류기를 제거한다는 뜻이다. 그 위에 가위, 바위, 보 3개 클래스를 위한 새 분류기를 붙인다.

6. QAT란 무엇인가?

QAT는 Quantization Aware Training의 약자다. 한국어로는 양자화 인식 학습이라고 부른다.

일반적으로 딥러닝 모델은 float32 숫자로 계산한다. 하지만 온디바이스에서는 int8 같은 작은 숫자 형식으로 바꾸면 모델 크기와 연산량을 줄일 수 있다. 이 변환을 양자화라고 한다.

항목 float32 모델 int8 양자화 모델
숫자 크기 32비트 8비트
모델 크기 큼 작음
추론 속도 상대적으로 느림 빠를 수 있음
정확도 위험 기준 변환 과정에서 손실 가능

QAT는 학습 중에 양자화 오차를 미리 흉내 낸다. 그래서 나중에 TFLite로 변환했을 때 정확도 손실을 줄이는 데 도움이 된다.

7. 코드에서 QAT가 적용되는 부분

def apply_quantization_to_layer(layer):
    if isinstance(layer, (keras.layers.Dense, keras.layers.Conv2D)):
        return tfmot.quantization.keras.quantize_annotate_layer(layer)
    return layer

annotated_model = keras.models.clone_model(
    model,
    clone_function=apply_quantization_to_layer,
)

qat_model = tfmot.quantization.keras.quantize_apply(annotated_model)

이 코드는 모든 레이어를 무조건 양자화하지 않는다. 주로 계산량이 큰 Dense와 Conv2D 레이어를 양자화 대상으로 표시한다. 초보자에게는 “중요한 계산 레이어에 양자화 훈련 장치를 붙인다”고 이해하면 된다.

8. TFLite 변환

학습된 QAT 모델은 마지막에 TFLite 형식으로 변환한다.

converter = tf.lite.TFLiteConverter.from_keras_model(model_best)
converter.optimizations = [tf.lite.Optimize.DEFAULT]
tflite_model = converter.convert()

.tflite 파일은 라즈베리파이에서 실행할 최종 모델이다. Keras 모델은 학습과 실험에 좋고, TFLite 모델은 배포와 추론에 적합하다.

핵심 정리

경량화는 단순히 모델을 작게 만드는 작업이 아니다. 제한된 장치에서 정확도, 속도, 메모리 사용량을 함께 맞추는 온디바이스 AI의 핵심 설계 과정이다.
